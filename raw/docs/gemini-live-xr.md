# Gemini Live API for Android XR

Source: https://developer.android.com/develop/xr/jetpack-xr-sdk/gemini-live
Fetched: 2026-05-21

## Overview
Bidirectional audio streaming via Firebase AI Logic. Handles both input and output.

## Requirements
- Persistent internet connection
- Incurs cost
- Limited concurrent connections per project

## Components
- LiveGenerativeModel — configuration
- LiveSession — bidirectional audio streaming
- FunctionDeclaration — custom agentic capabilities

## vs TTS/ASR
+ Seamless conversational flow
+ Lower latency
+ Agentic capabilities
- Requires internet
- Costs money
- Not suitable for offline critical communication
