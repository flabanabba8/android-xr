# Create an Android XR Project

Fetched from: https://developer.android.com/develop/xr/jetpack-xr-sdk/create-project
Date: 2026-05-21

## New Project

1. Open Android Studio (Canary)
2. File → New → New Project (or New Project from welcome screen)
3. Templates list → XR
4. Choose template for target device:
   - XR Headsets — immersive
   - Wired XR Glasses — immersive
   - Audio & Display Glasses — augmented
5. Click Next, name project, click Finish

## Adding XR to Existing Apps

1. Open project in latest Canary build
2. Set up Jetpack XR SDK dependencies
3. For immersive: follow "Bring your Android app into 3D with XR"
4. For augmented: follow "Start building for audio/display glasses"

## Status

SDK is Developer Preview 4. Report issues via support page.
