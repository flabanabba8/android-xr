# XR Visual Design Guidelines

Source: https://developer.android.com/design/ui/xr/guides/visual-design
Fetched: 2026-05-21

## Scale
- 0.868 dp-to-dmm conversion
- Optimal launch: 1.75m from user
- FOV: 41° recommended
- 1m real = 1m XR

## Typography
- Min font size: 14dp
- Min weight: normal
- Use sentence case, short line lengths
- Allow text scaling for accessibility
- Brighter text values for wired XR glasses
- Avoid all-caps, text on moving objects

## Target Sizes
- Recommended: 56×56dp, min 48×48dp
- Spacing: 8dp minimum between targets

## Color (Wired Glasses)
- Additive display: black = transparent
- Brighter colors = more visible/solid
- Darker colors = more transparent
- High-contrast bright colors for UI visibility
- Dimming lenses increase solidity

## Testing
- Motion sickness, smooth transitions, stable frame rate
- Passthrough in real-world settings
- Varying lighting (bright + dim)
- Text readability at different distances
- User testing with varied XR experience
