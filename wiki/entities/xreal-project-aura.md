# XREAL Project Aura

XREAL's wired XR glasses, launching before end of 2026. Part of the Android XR ecosystem and provided as dev kits through the [[catalyst-program]].

## Key Facts

- **Type:** Wired XR glasses (tethered)
- **Platform:** Android XR
- **SDK support:** Jetpack XR SDK, Unity, Godot, Unreal Engine
- **Developer access:** Early access through Catalyst Program
- **Launch:** Before end of 2026, early access for developers
