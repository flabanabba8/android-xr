# Android Studio for XR

Use the latest **Canary build** of Android Studio for XR development. Other versions may not include XR tools.

## XR-Specific Features

- **XR project templates** — pre-configured for headsets or glasses
- **Android XR Emulator** — test without physical hardware
- **Layout Inspector** — debug spatial UI layouts
- **AVD creation** — virtual devices for XR headsets and glasses
- **Composable Previews** — iterate on spatial UI in the IDE
- **Jetpack AI Copilot** — AI assistance for XR development
