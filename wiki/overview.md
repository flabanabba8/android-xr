# Android XR Wiki — Overview

This wiki covers Android XR development using the Jetpack XR SDK, targeting the Android XR Developer Catalyst Program grant.

## Coverage

- **SDK Architecture**: Jetpack XR SDK components — Compose for XR, SceneCore, ARCore, Glimmer, Projected
- **Device Types**: XR headsets, wired XR glasses (Project Aura), audio/display glasses
- **Grant Program**: Catalyst Program eligibility, deadlines, application strategy
- **Development Workflows**: Getting started, immersive vs augmented development paths
- **Setup Guides**: Android Studio configuration, project creation, emulator setup
