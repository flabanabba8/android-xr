# Source: XR Visual Design Guidelines

**Source:** developer.android.com/design/ui/xr/guides/visual-design
**Fetched:** 2026-05-21

## Key Takeaways

1. **Scale**: 0.868 dp-to-dmm, optimal 1.75m launch distance, 41° FOV
2. **Typography**: min 14dp font, normal+ weight, sentence case, short lines, allow text scaling
3. **Wired glasses color model**: additive display — black=transparent, brighter=more solid, darker=more transparent
4. **Target sizes**: 56×56dp recommended, 48dp minimum, 8dp spacing
5. **Accessibility**: hover/focus states, text descriptions, accessible color combinations, clear letterforms
6. **For captioning**: high-contrast bright text on wired glasses, configurable font size, avoid text on moving objects
